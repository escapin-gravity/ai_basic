# 🚀 AI Basic  

### 📖 Overview  
**AI Basic** 레포지토리는 PyTorch를 기반으로 인공지능과 딥러닝의 핵심 개념 및 구현을 학습하기 위한 프로젝트입니다.  
단계별 학습 자료와 실습 예제들을 통해 기초부터 고급 주제까지 다룹니다.

---

### 🧠 Topics Covered  
다음은 레포지토리에서 다루는 주요 주제들입니다:  

1. **🔹 PyTorch Basics**  
   - 텐서 생성 및 조작  
   - 데이터 로딩 및 전처리  

2. **🔹 Autograd (자동미분)**  
   - 자동 미분 메커니즘 이해  
   - 그래디언트 계산  

3. **🔹 Gradient Descent (경사하강법)**  
   - 손실 함수 정의  
   - 모델 파라미터 최적화  

4. **🔹 Optimization (최적화)**  
   - 다양한 최적화 알고리즘 (SGD, Adam, RMSprop 등)  

5. **🔹 Regression (회귀)**  
   - 선형 회귀, 다항 회귀 구현  
   - 모델 훈련 및 평가  

6. **🔹 Custom Models**  
   - 사용자 정의 모델 구성  
   - PyTorch 클래스 활용  

7. **🔹 Deep Neural Networks (DNN)**  
   - 다층 신경망 설계  
   - 활성화 함수 적용 및 최적화  

8. **🔹 Initialization / Batch Norm / Dropout**  
   - 가중치 초기화  
   - 배치 정규화 및 드롭아웃 활용  

9. **🔹 Convolutional Neural Networks (CNN)**  
   - 합성곱 신경망 구조 및 이미지 데이터 처리  

10. **🔹 Autoencoders**  
    - 데이터 압축 및 복원  
    - 딥러닝 기반 특성 학습  

11. **🔹 Generative Adversarial Networks (GAN)**  
    - Generator와 Discriminator 구조  
    - 생성 모델 학습  

12. **🔹 Recurrent Neural Networks (RNN, LSTM, GRU)**  
    - 순차 및 시계열 데이터 처리  
    - 다양한 RNN 아키텍처 적용  

13. **🔹 Pre-trained Models**  
    - 사전학습 모델 활용 (Transfer Learning)  
    - Fine-tuning을 통한 적응 학습  

14. **🔹 Early Stopping**  
    - 과적합 방지를 위한 조기 종료 전략  

15. **🔹 Tensor Models**  
    - 텐서를 활용한 고성능 모델 설계
